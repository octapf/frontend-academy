# Query keys, `staleTime` e invalidación

## Qué vas a entender

Las keys deben incluir todo lo que define el resultado: usuario, filtros, locale.

## Detalle

`staleTime` evita refetch agresivo; `invalidateQueries` alinea mutaciones con lecturas.
