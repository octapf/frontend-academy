# `next/image` y `next/link`

## Qué vas a entender

`Image` optimiza formatos y tamaños; definí `width`/`height` o `fill` para evitar CLS.

## Detalle

`Link` prefetch de rutas internas cuando el enlace entra en viewport; útil para UX de SPA.

## Práctica recomendada

- Anotá **un caso real** de tu repo donde este tema apareció (o debería haber aparecido).
- Si aplica, usá **DevTools** (red, consola, performance, layout) para conectar el concepto con lo observable.

## Errores comunes

- Olvidar `images.remotePatterns` y romper builds con dominios externos.
