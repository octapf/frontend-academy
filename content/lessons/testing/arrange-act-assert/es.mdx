# Arrange · Act · Assert

Un patrón simple para escribir tests claros:

- **Arrange**: preparás datos y estado
- **Act**: ejecutás la acción
- **Assert**: verificás el resultado

Esto hace que el test sea más fácil de leer y mantener.
