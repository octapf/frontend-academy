# Depurar con `screen`

## Qué vas a entender

`screen.debug()` imprime el DOM actual; combiná con `logRoles` para descubrir queries.

## Detalle

Si no encontrás rol, revisá si el componente es accesible de verdad.

## Práctica recomendada

- Anotá **un caso real** de tu repo donde este tema apareció (o debería haber aparecido).
- Si aplica, usá **DevTools** (red, consola, performance, layout) para conectar el concepto con lo observable.

## Errores comunes

- Copiar el snippet sin adaptar convenciones del equipo (nombres, capas, tests, accesibilidad).

## Recursos

- [MDN Web Docs](https://developer.mozilla.org/es/)
- Documentación oficial de React, TypeScript o tu bundler, según el tema.
