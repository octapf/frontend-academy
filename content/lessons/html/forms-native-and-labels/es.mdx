# Formularios nativos y etiquetas

## Qué vas a entender

El atributo `for` del label debe coincidir con el `id` del input; en listas dinámicas usá ids estables o `aria-labelledby`.

## Detalle

Los mensajes de validación conviene enlazarlos con `aria-describedby` para que el lector anuncie contexto.

## Práctica recomendada

- Anotá **un caso real** de tu repo donde este tema apareció (o debería haber aparecido).
- Si aplica, usá **DevTools** (red, consola, performance, layout) para conectar el concepto con lo observable.

## Errores comunes

- Simular un checkbox con `div` y `onClick` sin rol ni teclado.
