# Array.reduce for grouping

`reduce` lets you build an accumulated result while iterating an array.

## Typical case: groupBy

Turn a list into a dictionary by key:

- input: `[{team:"a"},{team:"b"},{team:"a"}]`
- output: `{ a: [...], b: [...] }`

In the exercise you’ll implement `groupBy(items, keyFn)`.
