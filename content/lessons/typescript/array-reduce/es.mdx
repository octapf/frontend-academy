# Array.reduce para agrupar

`reduce` permite acumular un resultado mientras recorrés un array.

## Caso típico: groupBy

Transformar una lista en un diccionario por key:

- input: `[{team:"a"},{team:"b"},{team:"a"}]`
- output: `{ a: [...], b: [...] }`

En el ejercicio vas a implementar `groupBy(items, keyFn)`.
