# `import type`

`import type { Foo } from "./foo"` imports **types only**: bundlers can erase them completely from emitted JS—handy to avoid circular value imports or heavy modules used only in annotations.

```ts
import type { User } from "./types";

export function greet(u: User) {
  return `Hello ${u.name}`;
}
```

## Re-exports

When you re-export a public API from an internal package, split **value exports** from **type exports** so consumers don’t drag compile-only modules into the client.
