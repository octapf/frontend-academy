# Performance: memoization

Before optimizing, **measure**. Still, there are common patterns:

- `React.memo` to avoid re-renders when props are stable
- `useMemo` for expensive computations
- `useCallback` to pass stable callbacks

Warning: memoizing everything can backfire (complexity, memory).
