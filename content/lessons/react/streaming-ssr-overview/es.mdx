# Streaming SSR (idea general)

## Qué vas a entender

El servidor puede empezar a enviar HTML antes de tener toda la página lista, mejorando el first paint.

## Detalle

En frameworks modernos el detalle varía; el concepto útil es **dividir trabajo** entre server y cliente.

## Práctica recomendada

- Anotá **un caso real** de tu repo donde este tema apareció (o debería haber aparecido).
- Si aplica, usá **DevTools** (red, consola, performance, layout) para conectar el concepto con lo observable.
